# Reckless.rs

Reckless is a plugin manager for core lightning that take care about all the configuration and installation of a plugin for your core lightning node.

But Reckless is much more, it is also a development tool for the people that want develop plugin for core lightning, providing plugin template for all
the official languages (for Reckless).
